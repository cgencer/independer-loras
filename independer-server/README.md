## Independer App Server

Docker is required.

### Commands

#### Run and Install

- `docker-compose up`

#### Status

- `docker ps -a`
- `docker compose ls`

#### Stop and Clean

- `docker compose stop`

- `docker stop independer-app independer-db`
- `docker rm independer-app independer-db`
- `docker rmi independer-app-img`
